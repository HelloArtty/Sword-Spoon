## 🌐 เล่นผ่านเว็บเบราว์เซอร์ (WebGL)

วิธีที่ง่ายที่สุด ไม่ต้องติดตั้งโปรแกรมใดๆ เพียงแค่มีอินเทอร์เน็ตและเบราว์เซอร์

### ขั้นตอน

1. เปิดเว็บเบราว์เซอร์ที่รองรับ (แนะนำ **Google Chrome** หรือ **Microsoft Edge**)
2. ไปที่ลิงก์: [https://play.unity.com/en/games/6a3b1d5a-23ae-4f11-aa37-7b9fbf7d1b1f/webgl-builds](https://play.unity.com/en/games/6a3b1d5a-23ae-4f11-aa37-7b9fbf7d1b1f/webgl-builds)
3. รอให้เกมโหลดจนครบ (ขึ้นอยู่กับความเร็วอินเทอร์เน็ต)
4. กด **Play** แล้วเริ่มเล่นได้เลย

### ข้อกำหนดเบื้องต้น

| รายการ | รายละเอียด |
|--------|-----------|
| เบราว์เซอร์ | Chrome 90+, Edge 90+, Firefox 88+ |
| การเชื่อมต่อ | อินเทอร์เน็ตความเร็ว 5 Mbps ขึ้นไป |
| RAM | 4 GB ขึ้นไป |
| WebGL | ต้องเปิดใช้งาน (เปิดโดยค่าเริ่มต้น) |

> **หมายเหตุ:** ไม่แนะนำให้เล่นบน Safari เนื่องจากอาจมีปัญหาเรื่องประสิทธิภาพ

## 🪟 ติดตั้งบน Windows

### ความต้องการของระบบ

| รายการ | ขั้นต่ำ | แนะนำ |
|--------|--------|-------|
| ระบบปฏิบัติการ | Windows 10 (64-bit) | Windows 11 (64-bit) |
| RAM | 4 GB | 8 GB |
| พื้นที่ฮาร์ดดิสก์ | 500 MB | 1 GB |
| การ์ดจอ | DirectX 11 | DirectX 12 |
| .NET | .NET Framework 4.x | .NET 6+ |

### ขั้นตอนการติดตั้ง

1. **ดาวน์โหลดไฟล์** โดยคัดลอกโฟลเดอร์ `Sword&SpoonWin` ไปวางในตำแหน่งที่ต้องการ เช่น `C:\Games\Sword-Spoon\`

2. **โครงสร้างไฟล์ที่ต้องมีครบ** ตรวจสอบให้แน่ใจว่าโฟลเดอร์มีไฟล์ต่อไปนี้ครบทุกไฟล์:
   ```
   Sword&SpoonWin/
   ├── Game-Test.exe               ← ไฟล์หลักสำหรับรันเกม
   ├── UnityPlayer.dll             ← ไลบรารี Unity (จำเป็น)
   ├── UnityCrashHandler64.exe     ← ตัวจัดการข้อผิดพลาด
   ├── Game-Test_Data/             ← ข้อมูลเกม (ห้ามลบ)
   └── MonoBleedingEdge/           ← Runtime ของ Mono (ห้ามลบ)
   ```

   > **สำคัญ:** ห้ามย้ายไฟล์ `Game-Test.exe` ออกจากโฟลเดอร์ เพราะจะทำให้เกมไม่สามารถหาไฟล์ข้อมูลได้

3. **เปิดเกม** ดับเบิลคลิกที่ไฟล์ `Game-Test.exe`

4. **กรณีมี popup เตือนความปลอดภัยจาก Windows:**
   - คลิก **"More info"** (ข้อมูลเพิ่มเติม)
   - คลิก **"Run anyway"** (เรียกใช้งานต่อไป)

5. **ปรับแต่งการตั้งค่า** เมื่อหน้าต่างตั้งค่าปรากฏขึ้น ให้เลือกความละเอียดหน้าจอและกราฟิกที่ต้องการ จากนั้นกด **Play**

### การสร้าง Shortcut บน Desktop

1. คลิกขวาที่ `Game-Test.exe`
2. เลือก **Send to → Desktop (create shortcut)**
3. จะได้ไอคอนเกมบน Desktop สำหรับเปิดได้สะดวก

## 🍎 ติดตั้งบน macOS

### ความต้องการของระบบ

| รายการ | ขั้นต่ำ | แนะนำ |
|--------|--------|-------|
| ระบบปฏิบัติการ | macOS 11 Big Sur | macOS 13 Ventura ขึ้นไป |
| ชิป | Intel 64-bit หรือ Apple Silicon (M1/M2/M3/M4) | Apple Silicon |
| RAM | 4 GB | 8 GB |
| พื้นที่ฮาร์ดดิสก์ | 500 MB | 1 GB |

### ขั้นตอนการติดตั้ง

1. **คัดลอกโฟลเดอร์** `Sword&SpoonMac` ไปวางในตำแหน่งที่ต้องการ เช่น `/Applications/` หรือโฟลเดอร์ใดก็ได้

2. **โครงสร้างไฟล์ที่ต้องมีครบ:**
   ```
   Sword&SpoonMac/
   └── Sword&Spoon.app    ← แอปพลิเคชันหลัก (ดับเบิลคลิกเพื่อเปิด)
   ```

3. **เปิดเกมครั้งแรก (สำคัญมาก)** — เนื่องจาก macOS มีระบบความปลอดภัย Gatekeeper ต้องทำตามขั้นตอนนี้:

   **วิธีที่ 1: คลิกขวา (แนะนำ)**
   - คลิกขวาที่ `Sword&Spoon.app`
   - เลือก **"Open"** (เปิด)
   - เมื่อมี dialog แจ้งเตือนปรากฏ คลิก **"Open"** อีกครั้ง
   - หลังจากนี้สามารถดับเบิลคลิกเปิดได้ตามปกติ

   **วิธีที่ 2: ผ่าน System Settings**
   - ไปที่ **System Settings → Privacy & Security**
   - เลื่อนลงมาดูส่วน Security
   - จะเห็นข้อความว่า `"Sword&Spoon" was blocked...` คลิก **"Open Anyway"**

   **วิธีที่ 3: ผ่าน Terminal** (สำหรับผู้ใช้ขั้นสูง)
   ```bash
   xattr -cr /path/to/Sword&SpoonMac/Sword&Spoon.app
   ```
   แล้วดับเบิลคลิกเปิดได้เลย

4. **Apple Silicon (M1/M2/M3/M4):** หากเกมสร้างมาสำหรับ Intel อาจต้องใช้ Rosetta 2:
   - macOS จะถามให้ติดตั้ง Rosetta อัตโนมัติ คลิก **"Install"** แล้วรอสักครู่
   - หลังจากนั้นเปิดเกมได้ตามปกติ

### การย้ายไปที่ Applications

สามารถลากไฟล์ `Sword&Spoon.app` ไปวางในโฟลเดอร์ `/Applications/` เพื่อให้ปรากฏใน Launchpad ได้เลย

## 🔧 แก้ปัญหาเบื้องต้น

### Windows

| ปัญหา | วิธีแก้ไข |
|-------|---------|
| เกมไม่เปิด / แจ้งว่า DLL หาย | ติดตั้ง [Visual C++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe) |
| หน้าจอดำ / ค้าง | อัปเดตไดรเวอร์การ์ดจอ |
| FPS ต่ำ | ลดความละเอียดและคุณภาพกราฟิกในหน้าตั้งค่าก่อนเริ่มเกม |
| Windows บล็อกการรัน | คลิก "More info" → "Run anyway" |
| เกม Crash | อย่าลบโฟลเดอร์ `Game-Test_Data` และ `MonoBleedingEdge` ออกจากโฟลเดอร์ `Sword&SpoonWin` |

### macOS

| ปัญหา | วิธีแก้ไข |
|-------|---------|
| "Cannot be opened because developer cannot be verified" | คลิกขวา → Open → Open หรือไปที่ Privacy & Security → Open Anyway |
| เกมไม่ตอบสนองบน Apple Silicon | ติดตั้ง Rosetta 2 ด้วยคำสั่ง: `softwareupdate --install-rosetta` |
| ไม่มีเสียง | ตรวจสอบการตั้งค่าเสียงใน System Settings → Sound |
| ความละเอียดผิดพลาด | ลบไฟล์ preferences ของเกมใน `~/Library/Preferences/` แล้วเปิดใหม่ |

### WebGL

| ปัญหา | วิธีแก้ไข |
|-------|---------|
| เกมโหลดช้า / ค้าง | รีเฟรชหน้าเว็บ หรือล้าง Cache ของเบราว์เซอร์ |
| ไม่มีเสียง | ตรวจสอบว่าเบราว์เซอร์ไม่ได้ Mute เว็บไซต์ (ไอคอนลำโพงใน Tab) |
| กราฟิกผิดปกติ | อัปเดตเบราว์เซอร์ หรือเปิดด้วย Chrome/Edge |
| เกมไม่รัน | ตรวจสอบว่า WebGL เปิดอยู่: พิมพ์ `chrome://gpu` ใน Chrome |
